# Go web app

Go Fiber + GORM + SQLite + Gothic + Templ + TailwindCSS

## Install

```bash
$ cp .env.sample .env
$ cd styles/ && pnpm install
```

## Run

* Start app:

    ```bash
    $ make
    ```

## Dev

* Build & watch CSS styles:

    ```bash
    $ make css
    ```

* Format go code:

    ```bash
    $ make fmt
    ```
